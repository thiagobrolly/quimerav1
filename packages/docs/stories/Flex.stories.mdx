# Flex

`Flex` é um componente com display flex por padrão e vem com várias props de estilo úteis.
Ele renderiza um elemento `div` por padrão, mas através da prop `as` pode ser renderizado qualquer elemento html.

## Import

```sh
  import { Flex } from '@paylivre/components';
```
