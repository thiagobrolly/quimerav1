# Tooltip

O componente `Tooltip` é responsável por apresentar uma mensagem de texto ao usuário manter o mouse por cima

## Import

```sh
  import { Tooltip } from '@paylivre/components';
```

## Basic

O `Tooltip` espera receber um `trigger` com o texto da primeira camada e um `children` com o texto apresentado ao usuário manter o mouse sobre o componente.
