# Button

O componente `Button` é utilizado para permitir que o usuário possa realizar ações dentro da aplicação.

## Import

```sh
  import { Button } from '@paylivre/components';
```

## Variations Theme

O `Button` pode receber a prop `variant`. As variações de tema podem criar botões preenchidos, contornados ou apenas o texto.

<Canvas>
  <Story name="Thema">

    </Button>
    <Button m="0.5rem" variant="standard">
      Standard
    </Button>
    <Button m="0.5rem" variant="outline">
      Outline
    </Button>
    <Button m="0.5rem" variant="approve">
      Approve
    </Button>
    <Button m="0.5rem" variant="danger">
      Cancel
    </Button>
  </Story>
</Canvas>

## Size Variations

O `Button` pode receber a prop `size`. As variações de tamanho podem criar botões pequenos, medios, grandes e tamanho total disponível.
