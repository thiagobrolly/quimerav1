# Heading

O componente `Heading` é usado para renderizar elementos de cabeçalho HTML semânticos.

## Import

```sh
  import { Heading } from '@paylivre/components';
```

## Basic

<Canvas>
  <Story
    name="Default"
    args={{
      children: 'Heading',
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>

## Variations Theme

O `Heading` renderiza uma tag `<h1>` por padrão, mas pode receber qualquer outra tag de cabeçallho através da props `as`.

<Canvas>
  <Story name="as">
    <Heading as="h1">Olá, sou um h1</Heading>
    <Heading as="h2">Olá, sou um h2</Heading>
    <Heading as="h3">Olá, sou um h3</Heading>
    <Heading as="h4">Olá, sou um h4</Heading>
    <Heading as="h5">Olá, sou um h5</Heading>
    <Heading as="h6">Olá, sou um h6</Heading>
  </Story>
</Canvas>
